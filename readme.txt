== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/aio-calculators-by-adittaw` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to 'AIO Calculators' settings page to view and manage the available calculators and their shortcodes.

== Frequently Asked Questions ==

= How do I add a calculator to my site? =

Once the plugin is activated, you can use the provided shortcodes to embed calculators into your posts, pages, or widgets.

= Can I customize the appearance of the calculators? =

The appearance of the calculators can be customized by adding custom CSS to your theme's stylesheets.
